# 微信公众号文章下载器（chrome浏览器插件）

一个Chrome扩展程序，用于将当前正在浏览的微信公众号文章下载为Markdown格式，并支持图片和视频资源的下载。

# 主要功能

- 将微信公众号文章转换为Markdown格式
- 下载文章中的图片和视频资源
- 将所有内容打包成ZIP文件
- 保持原文排版和布局

# 使用方法

1. 在Chrome浏览器中安装扩展
2. 打开微信公众号文章页面
3. 点击扩展图标，选择下载选项
4. 等待下载完成
